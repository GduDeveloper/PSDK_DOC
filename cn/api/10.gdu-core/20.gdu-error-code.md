错误码的头文件为`gdu_error.h`，本文档描述了错误码相关的宏定义和结构体的关键信息以及使用方法。

### 宏定义

- 错误模块索引值偏移量

```C
#define GDU_ERROR_MODULE_INDEX_OFFSET  32u
```

- 错误模块索引值掩码

```c
#define GDU_ERROR_MODULE_INDEX_MASK    0x000000FF00000000u
```

- 原始错误码偏移量

```c
#define GDU_ERROR_RAW_CODE_OFFSET      0u
```

- 原始错误码掩码

```c
#define GDU_ERROR_RAW_CODE_MASK        0x00000000FFFFFFFFu
```

- 错误码生成宏

```c
#define GDU_ERROR_CODE(moduleIndex, rawErrCode) \
((uint64_t)0 | \
((((uint64_t)(moduleIndex)) << (GDU_ERROR_MODULE_INDEX_OFFSET)) & (GDU_ERROR_MODULE_INDEX_MASK)) | \
((((uint64_t)(rawErrCode)) << (GDU_ERROR_RAW_CODE_OFFSET)) & (GDU_ERROR_RAW_CODE_MASK)))
```

- 操作返回值

  - 操作成功

  ```c
  #define GDU_RETURN_CODE_OK                 GDU_ERROR_SYSTEM_MODULE_CODE_SUCCESS
  ```

  - 空间分配错误

  ```c
  #define GDU_RETURN_CODE_ERR_ALLOC          GDU_ERROR_SYSTEM_MODULE_CODE_MEMORY_ALLOC_FAILED
  ```

  - 操作超时

  ```c
  #define GDU_RETURN_CODE_ERR_TIMEOUT        GDU_ERROR_SYSTEM_MODULE_CODE_TIMEOUT
  ```

  - 对象不存在

  ```c
  #define GDU_RETURN_CODE_ERR_NOT_FOUND      GDU_ERROR_SYSTEM_MODULE_CODE_NOT_FOUND
  ```

  - 超出指定的范围

  ```c
  #define GDU_RETURN_CODE_ERR_OUT_OF_RANGE   GDU_ERROR_SYSTEM_MODULE_CODE_OUT_OF_RANGE 
  ```

  - 参数错误

  ```c
  #define GDU_RETURN_CODE_ERR_PARAM          GDU_ERROR_SYSTEM_MODULE_CODE_INVALID_PARAMETER
  ```

  - 系统错误

  ```c
  #define GDU_RETURN_CODE_ERR_SYSTEM         GDU_ERROR_SYSTEM_MODULE_CODE_SYSTEM_ERROR
  ```

  - 系统繁忙

  ```c
  #define GDU_RETURN_CODE_ERR_BUSY           GDU_ERROR_SYSTEM_MODULE_CODE_BUSY
  ```

  - 暂不支持

  ```c
  #define GDU_RETURN_CODE_ERR_UNSUPPORT      GDU_ERROR_SYSTEM_MODULE_CODE_NONSUPPORT
  ```

  - 未知错误

  ```c
  #define GDU_RETURN_CODE_ERR_UNKNOWN        GDU_ERROR_SYSTEM_MODULE_CODE_UNKNOWN
  ```
